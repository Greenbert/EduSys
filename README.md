# EduSys
- Servlet和Jsp实现一个简单的教务管理系统

- 后台技术使用servlet、druid数据库连接池，前台页面使用jsp，jQuery，bootstrap

### 1. 开发环境
- idea 2022.1
- jdk 1.8
- tomcat 9.0.45
- mysql 8.0

### 2. 实现功能
- 登录(教师, 学生)
- 注册(教师, 验证码)
- 找回密码(教师, 学生)
- 记住登录状态
- 学生管理(增删改查)
- 成绩管理(修改, 导出excel)
- 课程安排（教师，学生）
- 上传文件（头像）
- 个人成绩(导出pdf)
